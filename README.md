# Nobel Prize Catalogue

Zadanko rekrutacyjne do BIT Web

## Jak uruchomić?

1. git clone https://github.com/tudde/nobel-prize-catalogue.git
2. npm install
3. npm start
4. http://localhost:3000 w dowolnej przeglądarce
5. profit!
